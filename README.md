# アルゴリズム・トレーサー

基本情報技術者試験（FE）の科目B（アルゴリズム）を学習する初心者のための、ブラウザ完結型のPythonコード実行・トレースツールです。

## 特徴

- **ステップ実行**: Pythonコードを1行ずつ実行し、プログラムの流れを理解できます
- **変数の可視化**: 各ステップでの変数の値をリアルタイムで確認できます
- **ブラウザ完結**: Pyodideを使用し、サーバー不要でブラウザ上で完結します
- **直感的なUI**: コードエディタと変数モニターを並べて表示し、学習しやすいレイアウト

## 技術スタック

- **フレームワーク**: Next.js 15 (App Router)
- **言語**: TypeScript, React
- **スタイリング**: Tailwind CSS
- **コード実行**: Pyodide (ブラウザ上でPythonを実行)
- **エディタ**: Monaco Editor

## セットアップ

### 前提条件

- Node.js 18.17以上
- npm または yarn

### インストール

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## 使い方

### 基本操作

1. **コード編集**: 左側のエディタでPythonコードを編集します
2. **実行**: 「実行」ボタンをクリック、または `Ctrl/Cmd + Enter` でコードを実行します
3. **ステップ実行**:
   - 「次へ」ボタン（または右矢印キー）で次のステップに進みます
   - 「前へ」ボタン（または左矢印キー）で前のステップに戻ります
4. **変数確認**: 右側のパネルで現在のステップでの変数の値を確認できます
5. **リセット**: 「リセット」ボタンで実行状態をクリアします

### キーボードショートカット

- `Ctrl/Cmd + Enter`: コードを実行
- `→` (右矢印): 次のステップへ
- `←` (左矢印): 前のステップへ

### サンプルコード

初期表示されているサンプルコード:

```python
# 1から10までの合計を求める
total = 0
for i in range(1, 11):
    total = total + i
    print(f"{i}までの合計: {total}")

print(f"最終結果: {total}")
```

### 学習のポイント

1. **ループ処理の理解**: for文やwhile文での変数の変化を追跡
2. **条件分岐**: if文での実行パスの確認
3. **変数のスコープ**: 変数がいつ作成され、どう変化するかを確認
4. **アルゴリズムの動作**: ソートや探索アルゴリズムの動きを1ステップずつ確認

## 制限事項

- Pyodideの制約により、一部のPythonライブラリは使用できません
- ファイルI/O、ネットワーク通信などのシステム操作は制限されています
- 大規模なコードや無限ループには注意が必要です
